
# Generative-AI-Chat-interface-using-Streamlit-and-Gemini-1.5-flash

### Dependencies

```.txt
    streamlit
    google-generativeai
    python-dotenv
```
### To install the dependencies run the following command:

```bash
    pip install -r requirements.txt
```

### To run the app:

```bash
    streamlit run app.py
```
